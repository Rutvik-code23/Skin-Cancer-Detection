Design a minimalistic and professional Doctor Dashboard UI for a Skin Cancer Detection and Classification screening tool used by dermatologists in clinics or hospitals.

The interface should look clean, trustworthy, medical-grade, and easy for doctors to use quickly during patient screening.

Design Style

Minimalistic medical interface

Professional healthcare aesthetic

Clear hierarchy and readable typography

Plenty of white space

Soft rounded cards and subtle shadows

No gradients — use solid colors only
